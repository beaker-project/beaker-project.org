Beaker in a box

This kickstart and associated scripts can be used to install and configure a complete working Beaker environment, including three virtual guests to act as test systems. You will need a spare system capable of KVM virtualization with at least 4GB of RAM.

http://beaker-project.org/in-a-box/beaker.ks
